What I care about

I like projects where constraints create personality: a strong concept, clean execution, and a touch of humor. I’m happiest when the interface teaches you how to use it without explaining itself.

  • Prototype-first thinking (ship the idea, then refine the edges)
  • Accessible by default (semantic HTML, keyboard support, readable type)
  • Performance-minded (static when possible, minimal JS when needed)
